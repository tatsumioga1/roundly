Radius

Make sharp monitor edges feel at home with Windows 11.

Download View source

Quiet by design

Designed to disappear into Windows.

Click-through overlays

Radius draws tiny corner masks without moving, resizing, or clipping your apps.

System tray control

Open settings, toggle the effect, set startup behavior, or exit from the notification area.

Low resource use

No network service, no telemetry, no input hooks, and no constant background polling.

For sharp monitors

A tiny visual fix for a modern desktop.

Radius is built with WinUI 3 and packaged as MSIX, so it behaves like a native Windows app while staying simple enough to trust.
